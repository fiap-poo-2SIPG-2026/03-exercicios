# Exercício 01
O banco digital **NeoBank** está desenvolvendo a primeira versão de seu sistema interno para 
gerenciar contas correntes. Ao abrir uma conta pelo aplicativo, o sistema precisa registrar as 
seguintes informações:  
- Nome do correntista (titular da conta)  
- Número da conta (gerado pelo banco)  
- Saldo da conta  
Toda conta corrente pertence a um único correntista e possui um número exclusivo. No 
momento da abertura, a conta será iniciada com saldo zero.
Além de armazenar essas informações, o sistema deve permitir operações básicas de 
movimentação financeira
